Contest rules

The contest consists of multiple games, with participants scoring the most points, moving on to the next round.

A game consists of 10-20 questions, with a specified time limit either for the whole game or for each question. For each correct answer you get 1 point. The answer must consist of two parts:

  • The answer text (usually a few words only)
  • One or more links to the supporting pages that should explicitly contain this information.

An answer is considered correct if there is a web page containing your answer, and you provided its URL.

An answer is considered incorrect if:

  • One of the fields above is missing
  • The provided page does not contain the answer
  • The answer is too vague or does not address the question directly
  • The web page is manipulated by the participant (manually or automatically), to contain the answer.
  • The answer was not found through our search interface (logs will be checked to confirm that our interface was used).

The answer correctness will be checked by multiple human judges, with final arbitration reserved by the contest organizers.
